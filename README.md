# Weimo

**venv instructions:**

Windows:

```
py -3.13 -m venv .venv # Creating a venv called .venv
.venv\Scripts\activate # Activating venv '.venv'
pip install -r requirements.txt # Install all requirements into venv
deactivate # Turns off venv
```

Linux / macOS:

```
python3 -m venv .venv # Creating a venv called .venv
source .venv/bin/activate # Activating venv '.venv'
pip install -r requirements.txt # Install all requirements into venv
deactivate # Turns off venv
```

> Note that different folders might have different dependencies
